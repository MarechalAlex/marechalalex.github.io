Module Vpl__Debugger

This module handles debug traces for all other modules. The level of details in traces can be tuned independantly for each module, during execution (see DebugTypes.levelT). For instance, if you want to see only the Title level for module Proj, type Proj.Debug.enable [Title]. For readability, each module is associated to a color, available only in on-the-fly mode.

val enable : unit -> unit

Enables debug traces for all modules.

val disable : unit -> unit

Disables debug traces for all modules.

val set_colors : unit -> unit

Set colors for each module

val print_enable : unit -> unit

Prints traces on-the-fly.

val print_disable : unit -> unit

Disable on-the-fly mode.

val reset_trace : unit -> unit
