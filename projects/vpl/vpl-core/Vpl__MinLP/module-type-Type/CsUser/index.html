val equalSyn : t -> t -> bool

Checks the syntactical equality between two constraints. x = 1 and x = 1 are syntactically equal, but not 2 * x = 2 and x = 1.

val inclSyn : t -> t -> bool

Check the syntactical inclusion of one constraint in another. The linear term of both constraints must be syntactically equal. x = 0 is included in x < 1, but not is 2 * x < 2.

type prop_t =
| Trivial

0 = 0 or 0 < 1 are Trivial

| Contrad

trivialy contradictory: e.g. 0 = 1 or 0 < -1

| Nothing

neither trivial nor trivialy contradictory

The type of properties the can be told by examining a constraint on its own.

val getCoefsFor : Vpl.Var.t option -> t list -> Vec.Coeff.t list

getCoefsFor mx l gathers the "coefficient" of mx in the constraints in l. If mx is None then the constants are gathered. If mx is Some x, the coefficients of variable x are gathered. The order of the numbers in the returned list matches that of the constraints in l.
