Module Vpl.Profile

This module implements a simple hand-made profiling tool. To use it, simply instantiate the fonctor Profile with a module D that contains a name. Example : module MyProfile = Profile(struct let name = "MyModuleName" end)

The "name" variable is used to prefix the name of functions in the report.
