Module Vpl.Domains

Available polyhedral domains. The user interface functor Vpl__UserInterface.Make must be provided with one of them. There are three levels of certification:

  • No certification: no certificate is produced
  • OCaml certification: Each operator of the domain produces a certificate, which is checked by an OCaml procedure.
  • Coq certification: In addition to guarantees offered by OCaml certification, certificates are here checked by a Coq-proved procedure.
