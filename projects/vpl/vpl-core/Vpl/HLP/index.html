Module Vpl.HLP

This module builds and solve the linear programming problem associated to the heuristic powerLTOne. Within this heuristic, we must choose one bound (either the upper or lower) per variable x_i. As some bounds may not exist if the input polyhedron is unbouded, we handle this using a type HOtypes.LPMaps.mapDetBound. Decision variables are called lpvars, and lpvars_i = 1 if we choose the upper bound of x_i.
