Module type Vpl__OraclePattern.OracleType

Interface of linearization Oracles. An Oracle has a modular design: strategies are encoded within Prayers, that are ordered in a list (prayers). Starting from a polynomial to treat, each prayer is tried one after the other. If it can be applied, the prayer updates the polynomial until it is 0.
