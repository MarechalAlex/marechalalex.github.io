Module Vpl__PSplx

This module defines the type of parametric simplex tableau. In VPL, the PLP is always use to express a combination of some input linear constraints. This specificity is thus put directly in the simplex tableau. Each column of the tableau is associated with an input constraint (with a certificate). Constraints of the problem are expressed as functions of these constraints.

module Debug : Vpl.DebugTypes.Type
module Cs = Vpl.Cstr.Rat
type decision_variable = int

Each column of the simplex tableau is associated with a decision variable. Therefore, decision variables are identified by their column position in the simplex tableau.

type var_set = int

Decision variables can be given a set of variables, with which they are allowed to pivot. By default, variables are given the set 1.

module VarMap : Stdlib.Map.S with type VarMap.key = decision_variable
type pivotT = (Cs.t * int * Vpl.Tableau.t) -> Cs.t

This type is used to replay pivots when adding new columns. It represents a pivot operation.

parameter cstr

the objective coefficient

parameter i_col

the index of the new column

parameter tab

the simplex tableau on which the pivot must be applied

returns

the new objective coefficient

val isFeasible : 'c t -> bool
returns

true is the basic solution is feasible, i.e. if all variables have a nonnegative value. The basic solution consists in setting all non-basic variables to zero and setting each basic variable to the right-hand side of the constraint it appears on.
